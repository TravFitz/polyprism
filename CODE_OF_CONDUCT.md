# Code of Conduct

This project follows the **Contributor Covenant v2.1**.

The full text — including community standards, enforcement responsibilities,
scope, and the four-tier enforcement ladder — lives at the canonical source:

**<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>**

## Reporting

If you experience or witness behaviour that violates the Covenant, please
contact the project maintainer at **travis.fitzgerald@zonos.com**. All
reports are handled confidentially and reviewed promptly.
